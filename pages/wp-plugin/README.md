# auditsana-website

## File map
| File | What it controls |
|---|---|
| pages/homepage.html | Homepage of auditsana.com |
| partials/header.html | Sitewide header (coming soon) |
| partials/footer.html | Sitewide footer (coming soon) |
| wp-plugin/auditsana-loader.php | WordPress plugin — upload to /wp-content/plugins/auditsana-loader/ |

## To update the site
1. Edit any file in pages/
2. Commit to main
3. Visit https://auditsana.com/?refresh=1 to clear cache
